Abstract

This is a manual for an

AutomataGroups

package, implementing in GAP basic functions and algorithms for groups generated by finite automata, contracting groups.

Acknowledgements

The development of this package was partially supported by National Science Foundation

Colophon

The project was originally started in 2000 mostly for personal use. It was gradually expadnig during consequent years, including both addition of new algorithms and simplification of user interface. It was used in the process of classification of groups generated by $3$-state automata over 2-letter alphabet.
